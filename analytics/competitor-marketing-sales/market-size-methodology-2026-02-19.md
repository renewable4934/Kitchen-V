# 01 Market Size Methodology

Дата фиксации методики: **19 февраля 2026**.

## 1) Цель расчета
Оценить объем рынка проданных кухонь в Ростове-на-Дону за 2024 и 2025 годы:
- в количестве клиентов;
- в денежном объеме.

## 2) Базовые формулы
- `N_kitchen_clients_y = N_primary_y * p_primary_y + N_secondary_y * p_secondary_y`
- `Market_value_y = N_kitchen_clients_y * AOV_y`

Где:
- `N_primary_y` — количество сделок по новостройкам в городе;
- `N_secondary_y` — количество сделок по вторичному рынку в городе;
- `p_primary_y` — доля покупателей первички, которые покупают новую кухню;
- `p_secondary_y` — доля покупателей вторички, которые заменяют кухню;
- `AOV_y` — средний чек покупки кухни.

## 3) Входные данные по первичке
- База по 2025: `11 900` сделок (bnMAP, материал от **30 декабря 2025**).
- Для 2024: обратный расчет из указанного в источнике прироста `+4.2%` в 2025 году.

## 4) Входные данные по вторичке (две ветки)
### Ветка A: city-level фрагмент
Используется доля первички в структуре сделок по Ростову-на-Дону:
- 2025: `53%`;
- 2024: `44%` (показатель за II полугодие, применен к году как proxy).

Формула восстановления вторички:
- `N_secondary_A = N_primary * (1 - share_primary) / share_primary`.

### Ветка B: proxy-оценка
Используется структура ЮФО (январь-май 2025):
- `66%` ДКП, `34%` ДДУ.

Формула:
- `ratio_sec_to_primary = 0.66 / 0.34`;
- `N_secondary_B = N_primary * ratio_sec_to_primary`.

### Базовое значение вторички
- `N_secondary_base = 0.7 * N_secondary_A + 0.3 * N_secondary_B`.

В таблице используются:
- `base_min` = ветка A;
- `base` = 70/30 blend;
- `base_max` = ветка B.

## 5) Сценарии вероятности покупки кухни
- `conservative`: `p_primary=0.70`, `p_secondary=0.15`
- `base`: `p_primary=0.82`, `p_secondary=0.25`
- `aggressive`: `p_primary=0.90`, `p_secondary=0.35`

## 6) Сценарии среднего чека (AOV)
AOV задан дискретно по сценарию:
- 2024: `220k / 300k / 390k` руб
- 2025: `240k / 330k / 430k` руб

И отдельно для базового диапазона:
- 2024: `base_min=270k`, `base_max=340k`
- 2025: `base_min=300k`, `base_max=380k`

## 7) Контроль качества расчета
- Проверка полного совпадения формул с `02_market_size_results_2024_2025.csv`.
- Проверка чувствительности: `base_min <= base <= base_max` по денежному объему.
- Проверка целостности: каждая строка результата имеет ссылочный источник в `sources_registry.csv`.

## 8) Ограничения
- Нет публичной официальной статистики именно по продажам кухонь в разрезе города.
- Часть параметров (p_primary, p_secondary, AOV) является модельными допущениями, вынесенными в `assumptions.md`.
