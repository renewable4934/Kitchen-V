# Execution Pack: 152 Hours

Пакет для операционного исполнения плана `22.02.2026-31.03.2026` по 4 часа ежедневно.

## Что использовать каждый день

1. Открой карточку текущей даты в `docs/execution/daily_cards/`.
2. Выполни 4 часовых блока с фиксацией факта и артефакта.
3. Обнови KPI и профильные трекеры в `data/ops/`.
4. Отправь клиентский daily-отчет по шаблону.

## Основные файлы

1. `docs/execution/PLAN_152H_2026-02-22_2026-03-31.md` — мастер-план.
2. `data/ops/Daily_Execution_Schedule_2026-02-22_2026-03-31.csv` — полный календарь 38 дней.
3. `data/ops/KPI_Daily_Sheet_2026-02-22_2026-03-31.csv` — ежедневные KPI.
4. `data/ops/Experiment_Register_2026Q1.csv` — учет A/B экспериментов.
5. `data/ops/Creative_Registry_2026Q1.csv` — учет креативов и эффективности.
6. `data/ops/SEO_Tracker_2026Q1.csv` — трекинг SEO-работ.
7. `data/ops/AI_Delegation_Log_2026Q1.csv` — журнал делегирования AI.
