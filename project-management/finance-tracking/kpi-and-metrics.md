# KPI AND METRICS

Актуально на: **19 февраля 2026**

## 1. Главные KPI

1. Лиды в месяц (целевые): количество заявок, пригодных для обработки.
2. Конверсия сайта в лид: `лиды / посетители * 100%`.
3. CPL (Cost Per Lead): стоимость 1 лида.
4. Дозвон: доля лидов, с которыми удалось связаться.
5. Квалификация: доля лидов, соответствующих целевой аудитории.
6. Конверсия в замер/просчет: доля лидов, дошедших до следующего этапа.

## 2. Базовые ориентиры на запуск

1. Конверсия сайта в лид: `5-9%`.
2. Дозвон: `85%+`.
3. Квалификация: `60%+`.
4. Доход до замера/просчета: `35%+`.
5. SLA первого ответа: до `5 минут`.

## 3. Маркетинговые метрики (каналы)

1. CTR (Click Through Rate): кликабельность объявления.
2. CPC (Cost Per Click): стоимость клика.
3. CPM (Cost Per Mille): стоимость 1000 показов.
4. CR в лид (Conversion Rate): конверсия из клика в заявку.

## 4. Воронка (pipeline)

1. `new_lead`
2. `contacted`
3. `estimate_sent`
4. `measurement_booked`
5. `won/lost`

Для каждого этапа считаем:

1. Конверсию перехода на следующий этап.
2. Среднее время перехода.

## 5. Минимальный недельный отчет

1. Потрачено на рекламу.
2. Лидов всего и по каналам.
3. CPL по каналам.
4. Дозвон и квалификация по каналам.
5. Топ-3 проблемы и план исправления.
