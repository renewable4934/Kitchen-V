# WORK RULES

Актуально на: **19 февраля 2026**

## 1. Принципы работы

1. Все решения принимаются на основе данных, а не ощущений.
2. Любая гипотеза сначала тестируется на небольшом бюджете.
3. Изменения фиксируются письменно (что изменили, зачем, какой эффект).
4. Приоритет — прибыль и качество лидов, а не только объем заявок.

## 2. Правила по задачам

1. Каждая задача должна иметь: цель, срок, ответственного, критерий успеха.
2. Без владельца задача не считается активной.
3. Задачи без измеримого результата не запускаются.

## 3. Правила по рекламным запускам

1. Запуск только с UTM-метками.
2. Запуск только при рабочем трекинге событий.
3. Нельзя масштабировать кампанию, пока нет подтверждения по качеству лидов.

## 4. Правила по лидам

1. Первый контакт с лидом — до 5 минут в рабочее время.
2. Если не дозвонились, обязателен повторный контакт.
3. Каждый потерянный лид должен иметь зафиксированную причину.

## 5. Правила по изменениям сайта

1. Все правки через список гипотез (что и почему тестируем).
2. Сильные изменения дизайна/оффера — только как A/B тест.
3. Нельзя одновременно менять много переменных, чтобы не потерять причинно-следственную связь.

## 6. Коммуникация

1. Еженедельный отчет обязателен.
2. Отчет содержит: лиды, CPL, конверсию, дозвон, квалификацию, выводы, следующий план.
3. Проблемы и риски выносятся сразу, не откладываются до конца месяца.
