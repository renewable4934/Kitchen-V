# Title: Kitchen Anytype Zone
**Purpose:** Человеческая зона проекта для материалов, которые синхронизируются с Anytype и помогают принимать решения.  
**Owner:** Вы.  
**Last updated:** 2026-03-05

## Что хранится здесь
1. `inbox/` — новые материалы из Anytype без финальной категоризации.
2. `decisions/` — решения и обоснования.
3. `summaries/` — краткие сводки для человека.
4. `glossary/` — материалы для простых расшифровок терминов.
5. `next-actions/` — следующие шаги и задачи.
6. `registry/` — технический реестр синхронизации (не редактировать вручную без необходимости).

## Важные правила
1. Синхронизация с Anytype применяется только к `kitchen_anytype/`.
2. Удаления не выполняются: только архивирование.
3. При конфликте версий побеждает более новый timestamp.
